In this application you can:

  • Use tools for working
  • Working with frames(create new/ duplicate / delete)

  • Watching animation preview while working and in the full screen
  • Export the final result in .gif or .apng
  • All your actions are saved in local storage
  • Control your actions using keyboard shortcuts
